# Ageless Arch

Website for Ageless Arch.

Ageless Arch is focused on keeping Arch Linux systems "age free. Avoiding identity-based restrictions, tracking, and data collection where possible.

The project includes:
- AUR packages (`agelessd`, `agelessd-flagrant`)
- Forks of core components (systemd, xdg-desktop-portal)
- Planned custom repositories and install media

This site is currently under development.
